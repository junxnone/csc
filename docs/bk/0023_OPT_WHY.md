# 程序性能优化的意义

  - 处理器架构的发展，性能提升难度变大(主频/存储器)
      - 单核 -\> 多核 -\>众核 -\> 异构 -\> 专用
      - 指令级并行: 指令流水/多发射/乱序执行
      - 数据级并行: SIMD
